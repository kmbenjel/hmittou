# Data Assets

This directory keeps durable source files that are useful beyond the web reader itself.

## `hmittou-youtube-ar.srt`

Arabic creator captions for the YouTube video `NAUppGCebzY`.

Why it is stored here:

- It is the recoverable source for the published YouTube caption track.
- It preserves the current shatr-by-shatr timing against the existing audio.
- It keeps the repeated audio shatrs as separate cues, matching the recitation.
- It can be reused if the video is re-rendered, reuploaded, or restored.

What it is for:

- Upload it in YouTube Studio as Arabic subtitles with timing.
- Use it as the baseline if `index.html` poem text is corrected again.
- Keep the timing unless the audio changes; update only cue text for text-only corrections.

Last synced with the published YouTube Arabic captions on 2026-06-08.
